# പതിവായി കാണുന്ന അക്ഷരതെറ്റുകളുടെ പട്ടിക

മലയാളം എഴുതുമ്പോൾ പലപ്പോഴും തെറ്റായി പ്രയോഗിച്ചുകാണാറുള്ളതും ഭാഷയിലെ പ്രചാരം കൊണ്ട് ഇനിയും സാധുത വന്നിട്ടില്ലാത്തതുമായ പദങ്ങളുടെയോ പദഭാഗങ്ങളുടെയോ പട്ടിക.
